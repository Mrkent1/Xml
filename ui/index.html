THUẾ FORTRESS - XML Inspector

  1. Chạy tools/xml_inspect.ps1 để sinh reports/xml_diff.json (dry-run).
  2. Mở reports/xml_diff.json bằng trình xem JSON để đối chiếu REAL/FAKE.
  3. Chỉnh tools/config.json (chọn trường định danh, cấu hình Syncthing).
  4. Nếu muốn thử revert thật, đặt policy.dryRun=false rồi chạy tools/auto_revert.ps1.

Ghi chú: Đây là trang hướng dẫn tạm thời. GUI tick chọn trường sẽ được bổ sung sau khi chốt cấu hình.
